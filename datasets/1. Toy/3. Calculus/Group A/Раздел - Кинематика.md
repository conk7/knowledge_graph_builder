# Раздел - Кинематика

**Кинематика** (от греч. *kinema* — движение) — это раздел механики, изучающий геометрические свойства движения тел без рассмотрения причин, вызывающих это движение (массы и сил). Кинематика отвечает на вопрос "как движется тело?", в то время как динамика отвечает на вопрос "почему оно так движется?".

## Основные задачи кинематики
1.  **Описание положения тела:** Определение координат тела в любой момент времени. Для этого вводится система отсчета (тело отсчета, связанная с ним система координат и часы).
2.  **Определение траектории:** Нахождение линии, которую описывает тело при своем движении.
3.  **Определение скорости и ускорения:** Вычисление характеристик быстроты изменения положения тела и быстроты изменения его скорости.

## Основные понятия
*   **Материальная точка:** Тело, размерами которого в данной задаче можно пренебречь.
*   **Траектория:** Линия, вдоль которой движется тело.
*   **Путь ($s$):** Длина участка траектории, пройденного телом.
*   **Перемещение ($\Delta \vec{r}$):** Вектор, соединяющий начальное и конечное положение тела.
*   **Радиус-вектор ($\vec{r}$):** Вектор, проведенный из начала координат в точку, где находится тело.

## Математический аппарат
Кинематика неразрывно связана с математическим анализом. Основные кинематические величины определяются через производные:
*   Скорость — первая производная радиус-вектора по времени: $\vec{v} = \frac{d\vec{r}}{dt}$.
*   Ускорение — первая производная скорости (или вторая производная радиус-вектора) по времени: $\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$.

Обратная задача кинематики — нахождение положения тела по известной скорости или ускорению — решается с помощью интегрирования. Например, $\vec{r}(t) = \vec{r}_0 + \int_{t_0}^t \vec{v}(\tau) d\tau$.

## Типы движения
*   **Равномерное:** скорость постоянна по величине.
*   **Равнопеременное:** ускорение постоянно.
*   **Криволинейное:** движение по дуге, окружности и т.д. В этом случае ускорение раскладывается на тангенциальное (изменяет величину скорости) и нормальное (изменяет направление скорости).
