# Определение предела на языке ε-δ

**Определение предела на языке ε-δ** ("эпсилон-дельта") — это строгая математическая формулировка понятия предела, которая стала стандартом в современном математическом анализе. Она была разработана в XIX веке для того, чтобы избавиться от логических противоречий, связанных с использованием "бесконечно малых величин".

## Формулировка
Пусть функция $f(x)$ определена в некоторой проколотой окрестности точки $a$. Число $L$ называется пределом функции $f(x)$ при $x$, стремящемся к $a$, если для любого (сколь угодно малого) положительного числа $\varepsilon$ (эпсилон) найдется такое положительное число $\delta$ (дельта), что для всех $x$, удовлетворяющих условию $0 < |x - a| < \delta$, выполняется неравенство $|f(x) - L| < \varepsilon$.

Символическая запись:
$$\lim_{x \to a} f(x) = L \iff \forall \varepsilon > 0, \exists \delta > 0 : 0 < |x - a| < \delta \implies |f(x) - L| < \varepsilon$$

## Разбор определения
Это определение часто называют "игрой в эпсилон-дельта". Представьте, что скептик (или заказчик) дает вам допуск точности $\varepsilon$. Ваша задача — найти такой радиус $\delta$ вокруг точки $a$, чтобы значения функции для всех точек из этого радиуса (кроме самой $a$) попадали в коридор точности $L \pm \varepsilon$. Если вы можете найти такое $\delta$ для *любого* $\varepsilon$, который даст скептик, значит, предел действительно равен $L$.

Геометрически это означает: для любой горизонтальной полосы шириной $2\varepsilon$ вокруг прямой $y=L$ можно найти такой вертикальный интервал шириной $2\delta$ вокруг $x=a$, что график функции внутри этого вертикального интервала полностью лежит внутри горизонтальной полосы.

## Историческое значение
До появления этого определения математики (включая Ньютона и Лейбница) опирались на интуицию движения или на туманные понятия типа "исчезающих величин". Это приводило к парадоксам и ошибкам. Огюстен Луи Коши, Бернард Болцано и Карл Вейерштрасс перестроили анализ на строгий фундамент.
Язык $\varepsilon-\delta$ позволил:
*   Строго доказать основные теоремы анализа.
*   Четко определить понятие непрерывности.
*   Избежать парадоксов, связанных с бесконечностью (как в апориях Зенона).
*   Создать базу для развития более абстрактных разделов математики (топологии, функционального анализа).
