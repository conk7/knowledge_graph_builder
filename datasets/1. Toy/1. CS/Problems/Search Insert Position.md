Дан отсортированный массив уникальных целых чисел `nums` и целевое значение `target`. Необходимо вернуть индекс, если цель найдена. Если нет, вернуть индекс, где она была бы, если бы была вставлена по порядку.

Необходимо написать алгоритм со сложностью `O(log n)`.

**Пример:**  
Вход: `nums = [1,3,5,6]`, `target = 5`  
Выход: `2`

Вход: `nums = [1,3,5,6]`, `target = 2`  
Выход: `1`

решение:

Используем классический бинарный поиск. Поскольку массив отсортирован, мы можем делить область поиска пополам на каждом шаге.

```python
def searchInsert(nums, target):
    left, right = 0, len(nums) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
            
    return left
```

**Разбор:** Алгоритм работает за `O(log n)`, так как на каждой итерации пространство поиска сокращается вдвое. Память: `O(1)`.
